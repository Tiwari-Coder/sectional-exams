# Sectional Exam System

## Objective
Design a flow/process for a sectional exam system where an exam is divided into multiple sections.

## What is a Sectional Exam
A sectional exam divides the exam into multiple sections. Each section has its own questions and time limit.

## Exam Flow
1. User opens exam page
2. Exam instructions are displayed
3. User starts the exam
4. Section 1 starts with timer
5. User answers questions
6. Section is submitted
7. Next section is enabled
8. After all sections, exam is submitted
9. Result is generated

## Section-wise Flow
- Each section has fixed time
- User cannot skip sections
- Timer auto-submits section

## Result Logic
- Calculate marks section-wise
- Combine total marks
- Show pass/fail status
