# Sectional Exam Flow

1. Exam Page
   - Exam name
   - Total sections
   - Instructions

2. Start Exam
   - Section 1 unlocked
   - Timer starts

3. Section Process
   - Load questions
   - Save answers
   - Submit section

4. Section Completion
   - Lock completed section
   - Unlock next section

5. Final Submission
   - Submit all sections
   - Generate result
